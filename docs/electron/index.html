Electron!

To use features for Electron simply add:

1. .electron class to budy tag

2. Next, you can add title bar.

3. Close everything in .electron-main class

Code example:

<!DOCTYPE html>
<html lang="en">
                   
    <head>
        ...
        <link rel="stylesheet" href="cssvizo.css">
    </head>
                    
    <body class="electron">
        <div class="electron-titlebar bg-gray-1">
            <div class="app-name">ðŸ’¨ CSSVizo Docs</div>
            <div class="title">Electron</div>
            <div class="controls">
                <div><i class="bi bi-underscore">_</i></div>
                <div><i class="bi bi-square"></i></div>
                <div><i class="bi bi-x-lg"></i></div>
            </div>
        </div>
                    
        <div class="electron-main">
            <div class="container-fluid p-3">
                Content goes here...
            </div>
        </div>
    </body>
</html>
